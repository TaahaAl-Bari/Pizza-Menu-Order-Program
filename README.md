# Pizza-Menu-Order-Program
• The application should be asking for specific details from the user, such as their: Full name, phone number, and email address.  
• Enter the amount of pizza they want [maximum is 6, minimum is 1.]  
• Enter the size of pizza.  
• Enter the amount of toppings they want on the pizza (optional).  
• Do they want to request for a delivery.  
• A 10% discount should be provided, only if the total cost is below £20.  
• Total cost calculated and provided to the customer.  
• Order should be distributed to the chef.
